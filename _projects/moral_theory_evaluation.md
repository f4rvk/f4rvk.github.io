Moral Foundations Theory, the Theory of Dyadic Morality, the MAC model, and dual-process accounts dominate the landscape of moral psychology — but how well do they actually hold up as scientific theories rather than as compelling narratives? This project takes that question seriously by subjecting these frameworks to systematic evaluation using explicit theory assessment criteria (Form, Content, Function, Trajectory).

A central component is an LLM-assisted qualitative coding pipeline applied to approximately 5,700 open-ended survey responses. The pipeline integrates multiple coding passes (human, Claude, GPT) with BERTopic triangulation for thematic analysis, stratified sampling for validation, Cohen's kappa for inter-rater reliability, and a dedicated classifier for identifying AI-generated responses. The goal is to assess whether people's own moral reasoning maps onto the categories these theories propose — and where it doesn't.
